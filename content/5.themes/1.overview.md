A **theme** is a **Nuxt module** that defines the look and often **shell behaviour** of the desktop: background, taskbar, window chrome, boot flow, typography, PrimeVue presets, etc.

## Why “theme = OS”

Product metaphor: same **engine** (`@owdproject/core`), different **skin**. Switching themes changes layout and style, not the APIs apps use to open windows or expose commands.

## What not to duplicate

- Base **window behaviour** (z-order, stores) stays in **core**; the theme supplies **UI** (chrome) around content the core mounts.
- **Apps** stay visually neutral: they should not import “Win95-only” components if you want them to run under a GNOME-like theme.
