## Documentation site (`docs/`)

```text
docs/
├── content/
│   ├── index.md
│   ├── 1.getting-started/
│   ├── 2.authoring/
│   ├── 3.architecture/
│   ├── 4.apps/
│   ├── 5.themes/
│   ├── 6.setup/          # CLI, package linking, monorepo, desktop.config
│   ├── 7.internals/      # Boot, stores, kernel contract
│   └── 8.reference/
├── nuxt.config.ts
└── package.json
```

Numbered folder prefixes order the Docus sidebar. URLs drop the numeric prefix (e.g. **`/setup/package-linking`**).

## Client monorepo (runtime)

Lives in **`owdproject/client`** — not inside the docs tree:

```text
client/
├── desktop/              # Reference shell (desktop.config.ts, nuxt.config.ts)
├── packages/
│   ├── core/             # @owdproject/core
│   ├── kit-theme/
│   ├── kit-fs/
│   └── kit-explorer/
├── themes/               # gitignored — clone theme repos here for workspace dev
├── apps/                 # gitignored — clone app repos here
├── template/             # npm create owd scaffold (sync via desktop template)
└── docs/                 # this site
```

**External** (npm, not under **`packages/`**): **`@owdproject/module-fs`**, **`@owdproject/module-persistence`**, published themes/apps.

See [Monorepo and workspace](/setup/monorepo-workspace) and [Architecture overview](/architecture/overview).
