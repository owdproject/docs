## Prerequisites

- **Node.js** compatible with Nuxt 4
- **pnpm** (see **`packageManager`** in **`package.json`**)

## Clone and run

The documentation site lives in the **`docs/`** folder of the **client** monorepo (or a standalone **`owd-docs`** checkout if you mirror it separately).

```bash
cd client/docs
pnpm install
pnpm dev
```

Open the URL printed in the terminal (often **`http://localhost:3000`**). The project extends **Docus** (`nuxt dev --extends docus`).

## Production build

```bash
pnpm build
```

Output is under **`.output/`** for Node hosting or static deploy targets.

## Relationship to the OWD client

**Runtime code** (`desktop/`, **`packages/core`**, kits, themes) lives beside **`docs/`** in the same monorepo. When documenting APIs, cross-check:

- **`packages/core/DESKTOP_KERNEL.md`**
- [Kernel contract](/internals/kernel-contract) in this site

Embedding docs **inside** the themed desktop is described in [Documentation module](/architecture/docs-module).
